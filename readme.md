## ğŸ§  How It Works

- Tasks are represented as draggable cards.
- Columns act as drop zones.
- When a task is dragged and dropped, it is moved in the DOM.
- Task data (title & description) is synced to `localStorage`.
- On page reload, tasks are restored from stored data.
